# flutter_stock 一款 App 炒股软件
一个flutter  股票  K线 分时， 成交量 MACD KDJ 等功能的 ，现将代码奉献给大家；欢迎点赞👍

包含的功能有 分时线，K线，筹码分布等， 5档行情，成交量，KDJ，MACD ，普通坐标，对数坐标， 叠加功能；添加二级指标功能
